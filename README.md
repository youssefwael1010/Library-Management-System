# 📚 Library Management System (Console App)

A console-based **Library Management System** built with **C# (.NET 8)** as part of the **Full-Stack .NET Developer Training – Sprint 1**.

The project demonstrates the fundamentals of Object-Oriented Programming (OOP) by implementing a simple library where users can manage books, members, borrowing operations, searching, and reporting.

---

## 🚀 Features

* 📖 Add new books
* 👤 Register Regular and Premium members
* 🔄 Borrow books
* ✅ Return books
* 🔍 Search books and members
* 📚 View available books
* 📝 View member borrowing history
* ⏰ Generate late return reports
* 🌱 Seed data for testing
